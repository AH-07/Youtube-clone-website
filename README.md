# 📺 YouTube Clone

A front-end YouTube clone built with HTML and CSS. Developed as the main project during the [SuperSimpleDev](https://www.youtube.com/@SuperSimpleDev) HTML & CSS course.

## 🔗 Live Demo

👉 [View it live on GitHub Pages](https://ah-07.github.io/Youtube-clone-website/index.html)

## Features

- YouTube-style navigation bar with search, icons, and profile picture
- Collapsible sidebar with navigation links (Home, Explore, Subscriptions, etc.)
- Video thumbnail grid with channel info, view counts, and timestamps
- Video duration overlays on thumbnails
- Clickable thumbnails and channel icons that link to real YouTube videos
- Multi-page layout
